# SFML Game

## Install (from scratch)

### Ubuntu
```
sudo apt update
sudo apt install g++ cmake libsfml-dev git
git clone https://github.com/remiCzn/SFML-tiles.git
cd SFML-tiles
```

### Windows
- Install Visual Studio
- Clone the project:
```
git clone https://github.com/remiCzn/SFML-tiles.git
```
Open `SFML_Tiles_vs.sln`

## Build

```
mkdir build
cd build
cmake ..
make
```

Enjoy!
